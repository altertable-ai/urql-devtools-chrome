# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2026-03-16

### Added
- Initial release
- Cache explorer with entity list and detail view
- Real-time cache updates from `@urql/devtools` exchange
- Clickable entity references for navigation
- Support for Chrome Manifest V3
- Basic cache normalization following URQL's default key strategy
